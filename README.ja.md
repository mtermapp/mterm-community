# MTerm Community

*他の言語で読む: [English](README.md)*

ここは MTerm のコミュニティリポジトリです。

このリポジトリは、MTermに関するコミュニティでの議論、バグ報告、機能要望を集約するための場所です。

## イシューについて

バグを見つけた場合や機能の要望がある場合は、用意されているIssueテンプレートをご利用ください。

- [バグを報告する](../../issues/new?assignees=&labels=bug%2Ctriage&projects=&template=bug_report.yml&title=%5BBug%5D%3A+)
- [機能を要望する](../../issues/new?assignees=&labels=enhancement%2Ctriage&projects=&template=feature_request.yml&title=%5BFeature%5D%3A+)
