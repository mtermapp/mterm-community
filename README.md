# MTerm Community

Welcome to the MTerm Community repository!  
ここは MTerm のコミュニティリポジトリです。

This repository is a central place for community discussions, bug reports, and feature requests related to MTerm.  
このリポジトリは、MTermに関するコミュニティでの議論、バグ報告、機能要望を集約するための場所です。

## Issues / イシューについて

If you find a bug or have a feature request, please use the issue templates provided.  
バグを見つけた場合や機能の要望がある場合は、用意されているIssueテンプレートをご利用ください。

- [Report a Bug / バグを報告する](../../issues/new?assignees=&labels=bug%2Ctriage&projects=&template=bug_report.yml&title=%5BBug%5D%3A+)
- [Request a Feature / 機能を要望する](../../issues/new?assignees=&labels=enhancement%2Ctriage&projects=&template=feature_request.yml&title=%5BFeature%5D%3A+)
